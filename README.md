# 🚀 Smart Task Manager

A modern Task Management application built with Python, Streamlit, and SQLite.

## ✨ Features

- Add, Edit, Delete Tasks
- Mark Tasks as Completed
- SQLite Database
- Search Tasks
- Sort Tasks
- Filter Tasks
- Dashboard
- Progress Tracking
- Overdue Notifications
- Today's Tasks
- Upcoming Tasks
- Export to CSV
- Export to Excel

## 🛠 Technologies

- Python
- Streamlit
- SQLite
- Pandas
- OpenPyXL

## ▶️ Run the Project

```bash
streamlit run app.py
```
